✨ ברוכים הבאים!
זהו פרויקט קטן וקליל לניהול משימות יומיות (Todo List) שרץ על הקונסולה שלכם. הוא נכתב כולו ב-Java סטנדרטי ☕ וללא צורך בהתקנות מסובכות!

⚙️ מה בפנים?
לוגיקה: ניהול משימות בסיסי (הוספה, עדכון סטטוס, מחיקה, חיפוש).

שמירה: הנתונים שלכם נשמרים אוטומטית לקובץ בינארי בשם tasks.data.

טכנולוגיה: Java טהורה – אין Maven, אין Gradle, אין ספריות חיצוניות!

🚀 איך מריצים את זה? (הכי קל)
אם אתם עובדים עם IntelliJ IDEA, זה קלי קלות:

פתחו את הפרויקט ב-IntelliJ.

נווטו לתיקייה src ובחרו בקובץ Main.java.

לחצו על כפתור הריצה הירוק (▶) שליד הפונקציה main (או בחרו Run בתפריט העליון).

חלון הקונסולה ייפתח ותוכלו להתחיל להקליד את הבחירות שלכם מהתפריט! 🥳
💾 חשוב לדעת
הקובץ tasks.data נוצר אוטומטית בהרצה הראשונה. אל תמחקו אותו אם אתם רוצים לשמור את המשימות שלכם!

אם תנסו לקרוא את tasks.data, תראו רק סימנים מוזרים – זה בסדר, זה קובץ בינארי!
