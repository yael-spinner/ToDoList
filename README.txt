# הוראות הרצה - מערכת ניהול משימות (Todo List)

## 🎯 תיאור הפרויקט
יישום קונסולה (CLI) לניהול משימות.
שכבת הנתונים (Persistence) ממומשת באמצעות Java Serialization (Java סטנדרטי) ללא תלות בספריות חיצוניות.

## 🛠️ דרישות מוקדמות
* Java Development Kit (JDK) מותקן (מומלץ גרסה 8 ומעלה).
* סביבת פיתוח (IDE) כמו IntelliJ IDEA או Eclipse.

## 🚀 הוראות הרצה

### 1. הרצה מתוך IntelliJ IDEA (הדרך המומלצת)

1.  **פתיחת הפרויקט:** פתח את תיקיית הפרויקט (`ToDoList` במקרה זה) ב-IntelliJ.
2.  **סימון תיקיית מקור:** ודא שהתיקייה המכילה את קובצי ה-`.java` (במקרה זה, התיקייה `src`) מסומנת כתיקיית **"מקור" (Source Root)** בהגדרות המודול (Project Structure -> Modules).
3.  **איתור המחלקה הראשית:** נווט אל הקובץ `Main.java`.
4.  **הרצה:** לחץ קליק ימני על הקובץ `Main.java` ובחר **"Run 'Main.main()'"**, או לחץ על כפתור הריצה הירוק (▶) שליד הפונקציה `main`.
5.  **אינטראקציה:** התפריט והקלט/פלט יופיעו בחלון ה-Console בתחתית ה-IDE.

### 2. הרצה דרך שורת הפקודה (Terminal)

ניתן להריץ את הפרויקט גם מחוץ ל-IDE, לאחר קמפול:

1.  **נווט לתיקייה:** פתח את הטרמינל/CMD ונווט לתיקייה בה נמצאים קובצי ה-`.java` (למשל, התיקייה `src`).
2.  **קמפול:** קמפל את כל קובצי המקור:
    ```bash
    javac *.java
    ```
3.  **הרצה:** הרץ את המחלקה הראשית:
    ```bash
    java Main
    ```

## 💾 שמירה ונתונים
* הנתונים נשמרים אוטומטית בכל פעולת שינוי (הוספה, עדכון, מחיקה) לקובץ **`tasks.data`** שנוצר בתיקיית הבסיס של הפרויקט.
* הקובץ `tasks.data` הוא קובץ בינארי ואינו מיועד לקריאה על ידי אדם.
